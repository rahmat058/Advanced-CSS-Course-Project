# Advanced CSS Course Projects 

This workspace contains two practice projects built while learning advanced CSS and Sass techniques.

## 1. Natours
Natours is a modern travel booking landing page designed to showcase advanced styling with Sass, responsive layouts, clean typography, beautiful gradients, and smooth animations. It demonstrates how to build a polished UI using component-based CSS architecture.

## 2. Trillo
Trillo is a hotel booking web app interface focused on layout design and user experience. It highlights the use of Flexbox, reusable UI components, and modern styling patterns to create a clean and interactive dashboard-style page.

## Projects Included
- Natours: a fictional tour booking website
- Trillo: a fictional hotel booking application UI

## How to Run
Each project has its own folder with a local development setup:

- Natours: open the Natours folder and run the project using its available scripts.
- Trillo: open the Trillo folder and run the project using its available scripts.

These projects are great examples of modern front-end design and advanced CSS practice.
